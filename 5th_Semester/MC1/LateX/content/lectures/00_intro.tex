\begin{KR}{Learning Objectives}
By end of course, students should be able to:
\begin{itemize}
    \item Know design patterns in C and can apply them in their code
    \item Apply real-time programming techniques to process asynchronous events from several peripheral blocks
    \item Structure a firmware for a microcontroller system appropriately and partition it into modules
    \item Understand the interactions of software in C with hardware (memory, peripherals) and can apply this understanding to write efficient and stable software
    \item Develop, verify and debug microcontroller systems
    \item Systematically design and implement applications using state event techniques for several cooperating state machines
\end{itemize}
\end{KR}
