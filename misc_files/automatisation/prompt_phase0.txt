## PART B: CREATE 00_table_of_contents.tex

```
Create 00_table_of_contents.tex for [COURSE NAME] following layout_and_colours.sty.

Use: [course schedule, lecture titles, syllabus topics, textbook table of contents]

STRUCTURE:

\section{Table of Contents}

\begin{concept}{Course Structure Overview}\\
This summary covers [X] main topics across [Y] chapters:
\begin{itemize}
    \item Chapters 1-[N]: Core foundations
    \item Chapters [N+1]-[M]: Advanced topics
    \item Chapters [M+1]-[Y]: Applications and integration
\end{itemize}

\textbf{Total KRs:} [TBD - fill in after all chapters complete]
\textbf{Total Examples:} [TBD - fill in after all chapters complete]
\end{concept}

FOR EACH CHAPTER, CREATE THIS STRUCTURE:

\subsection{[Chapter Number]. [Chapter Title]}

\begin{definition}{Chapter [X]: [Title]}\\
\textbf{File:} \texttt{XX\_topic\_name.tex}

\textbf{Topics Covered:}
\begin{itemize}
    \item [Major topic 1]
    \item [Major topic 2]
    \item [Major topic 3]
\end{itemize}

\textbf{Key Concepts:} [One-line summary of chapter focus]

\textbf{Lecture Coverage:} Lecture(s) [X-Y]

\textbf{Related Assignments:} Assignment(s) [X-Y]
\end{definition}

IMPORTANT NOTE: Chapter = Section; Subsection = Subsection; Subsubsection = Topic within Subsection; Paragraph = Detailed point within subsubsection.

\begin{concept}{[Chapter X] Content Structure}
\textbf{[Chapter Title]} (\texttt{section})
\begin{enumerate}
    \item \textbf{[Subection Title]} (\texttt{subsection})
    \begin{itemize}
        \item [Subsection topic 1] (\texttt{subsubsection})
        \begin{itemize}
            \item [Detailed point 1] (\texttt{paragraph})
            \item [Detailed point 2] (\texttt{paragraph})
        \end{itemize}
        \item [Subsection topic 2] (\texttt{subsubsection})
    \end{itemize}
    \item \textbf{[Subection Title]} (\texttt{subsection})
    \begin{itemize}
        \item [Subsection topic]
    \end{itemize}
    \item ...
\end{enumerate}
\end{concept}

\subsubsection{[Chapter X] Kochrezepte (KRs) and Examples}

\begin{remark2}{KRs in Chapter [X]}\\
\begin{itemize}
    \item KR [X.1]: How to [Procedure name]
    \begin{itemize}
        \item Example [X.1.1]: [Brief description of example]
        \item Example [X.1.2]: [Brief description of example]
        \item ...
    \end{itemize}
    \item KR [X.2]: How to [Procedure name]
    \begin{itemize}
        \item Example [X.2.1]: [Brief description of example]
        \item Example [X.2.2]: [Brief description of example]
        \item ...
    \end{itemize}
    \item KR [X.3]: How to [Procedure name]
    \item ...
\end{itemize}

\textbf{Total KRs:} [Number]
\textbf{Total Examples:} [Number]
\end{remark2}

[REPEAT ABOVE STRUCTURE FOR ALL CHAPTERS]

\subsubsection{Complete Overview and Checklist}

\begin{itemize}
    \item [ ] Section 1: [Title] 
    \begin{itemize}
        \item [ ] Subection 1.1: [Title]
        \begin{itemize}
            \item [ ] Subsubsection 1.1.1: [Title]

                \item [ ] Paragraph: [Title]
                \item [ ] \important{KR: [Title]}
                \item ...
            \end{itemize}
            \item [ ] Subsubsection 1.1.2: [Title]
        \end{itemize}
        \item [ ] Subection 1.2: [Title]
    \end{itemize}
    \item [ ] Section 2: [Title]
    \item ...
\end{itemize}
\end{concept}

\begin{remark}
\textbf{Note:} This table of contents serves as both a navigation guide and 
a progress tracker. As you complete each chapter, you can check it off here.
\end{remark}

OUTPUT REQUIREMENTS:
- Filename: 00_table_of_contents.tex
- Start directly with \section{Table of Contents}
- List ALL chapters that will be in the final summary
- Include detailed structure outline for each chapter
- List all planned KRs for each chapter
- Use consistent formatting throughout
- Mark sections as [TBD] if details not yet known
- Show clear hierarchy: section → subsection → subsubsection → paragraph
```

## USAGE INSTRUCTIONS

**When to use:**
Initial Use of Part A and B:
- At the very beginning of a new course summary project
- After gathering course materials but before content extraction (Phase 1)
- Remark: During initial use we want to extract the contents of all course materials without changing anything, so the structure in Part B should reflect the original course structure as closely as possible.

Secondary Use of Part B:
- After completing Phase 1 (Content Extraction) to finalize the table of contents with improved structure and KR planning for summary
- Note: Verify new structure with me first (more details in 'Phase 1.5 - Structure Review')
- After completing Phase 2 (KR Creation) to update the table of contents with actual KR counts and any structural changes made during summarization

**Add to main.tex:**
```latex
\input{00_intro.tex}
\raggedcolumns
\pagebreak

\input{00_table_of_contents.tex}
\raggedcolumns
\pagebreak
```
